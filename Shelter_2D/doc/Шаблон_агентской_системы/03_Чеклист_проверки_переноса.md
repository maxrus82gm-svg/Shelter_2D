# Чеклист проверки переноса

Использовать после заполнения паспорта и подстановки placeholders.

## 1. Последовательность

- [ ] Прочитан `README_как_переносить.md`.
- [ ] Прочитан `MANIFEST_структуры.md`.
- [ ] Заполнен `00_Паспорт_проекта.md`.
- [ ] Проверена `02_Карта_переменных.md`.
- [ ] Выполнено `01_Правило_подгонки_под_новый_проект.md`.

## 2. Паспорт

- [ ] `PROJECT_NAME` заполнен.
- [ ] `PROJECT_ROOT` заполнен.
- [ ] `OBSIDIAN_VAULT` заполнен.
- [ ] `DOC_FOLDER` заполнен.
- [ ] `REPOSITORY_URL` заполнен или указан как `НЕ УКАЗАНО`.
- [ ] `USER_GIT_POLICY` заполнен.
- [ ] Неизвестные значения помечены как `НЕ УКАЗАНО`.

## 3. Placeholders

- [ ] В рабочих документах заменены placeholders, которые должны быть значениями проекта.
- [ ] В шаблонной папке placeholders оставлены только там, где документ остаётся шаблоном.
- [ ] Не осталось путей старого проекта.
- [ ] Не осталось названий старого проекта вне явно помеченных примеров.

## 4. Структура

- [ ] Структура соответствует `MANIFEST_структуры.md`.
- [ ] Есть папка `Шаблоны_документов`.
- [ ] Есть папка `Формы_отчётов`.
- [ ] Есть рабочий стартовый документ.
- [ ] Есть рабочий документ активной задачи.
- [ ] Есть правила Git/review.
- [ ] Есть правила документов.
- [ ] Есть правила субагентов.

## 5. Безопасность

- [ ] `.git` не менялся.
- [ ] `.obsidian` не менялась.
- [ ] Реальные конфиги агентов не менялись без разрешения.
- [ ] Git не использовался, если задача прямо не разрешила.
- [ ] Внешний агент не использовался вне scope.

## 6. Формат

- [ ] Markdown сохранён как UTF-8.
- [ ] Кириллица читается нормально.
- [ ] Файлы не записаны одной строкой.
- [ ] Нет буквальных `\n` вместо переносов строк.
- [ ] Windows-пути не экранированы двойными слэшами.

## 7. Отчёт

- [ ] Итоговый отчёт записан.
- [ ] Указаны созданные и изменённые файлы.
- [ ] Указано, использовался ли Git.
- [ ] Указано, использовались ли субагенты.
- [ ] Если были проблемы, они записаны в `PROBLEMS`.

# REVIEW REPORT

Форма используется только для ChatGPT-review после отчёта агента.

Агент-исполнитель не заполняет этот блок и не определяет актуальный последний commit для review.

Перед заполнением формы ChatGPT самостоятельно проверяет репозиторий или GitHub и определяет актуальный последний commit, который проверяется.

Commit message: <message>
Repository: <owner/repo>
Branch: <branch>

## Главное правило

Перед задачей агент читает:

```text
m:\GitHub\Shelter_2D\Shelter_2D\doc\Старт.md
m:\GitHub\Shelter_2D\Shelter_2D\doc\Задача_агенту.md
```

Затем агент читает только те правила, которые нужны текущей задаче.

Шаблоны в папке `Шаблоны_документов` не являются рабочими правилами обычных задач. Их используют только для переноса, обслуживания и развития агентской документации.
